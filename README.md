# On Screen
## Description
- **On Screen** is a web application that allows users to view currently playing movies 
- Users can also enter their ZIP code to display a map showing nearby movie theaters 

## Technologies Used
- HTML
- CSS
- JavaScript
- Bulma
- jQuery 
- Moment.js
